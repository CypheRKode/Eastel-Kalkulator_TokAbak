Cara deploy dekat GitHub Pages:

1. Upload semua file dalam folder ini ke repo GitHub:
   - index.html
   - style.css
   - script.js
   - folder images

2. Pastikan struktur folder kekal begini:

   index.html
   style.css
   script.js
   images/rm10.png
   images/phone.png
   images/whatsapp.png

3. Pergi ke GitHub repo > Settings > Pages.

4. Pilih:
   Source: Deploy from a branch
   Branch: main
   Folder: /root

5. Klik Save dan tunggu GitHub Pages publish link.
